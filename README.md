# SimWorld

一个用 Python + Pygame 构建的 2D 俯视像素风虚拟世界模拟器，拥有昼夜系统、城市道路网络、汽车交通与 NPC 社交关系机制。

## 功能特性

- **昼夜循环** — 草地亮度随时间动态变化，夜晚有深色遮罩效果
- **城市道路系统** — 10 条主干道 + 17 条支路构成完整路网，主干道含双向车道、人行道与黄色中心虚线
- **小汽车交通** — 约 50 辆彩色小汽车在主干道上双向行驶，循环穿越世界
- **NPC 行人** — 20 个 NPC 自主游走，只能在人行道和支路上行走，不会踏入主干道车行道
- **社交关系系统** — NPC 之间通过聊天积累好感度，分为陌生人 / 熟人 / 朋友三个等级
- **电话联系** — 成为朋友后可互相打电话，气泡显示手机图标
- **关系衰减 & 冲突** — 长时间不联系关系会衰减，偶尔触发争吵事件
- **差异化对话** — 不同关系等级触发不同聊天内容
- **角色面板** — 右侧面板显示所有 NPC 列表，点击查看详细信息（性别、年龄、职业、余额、好友）
- **小地图** — 右下角实时显示世界全局视图与当前视角位置

## 快速开始

**依赖**

```bash
pip install pygame
```

**运行**

```bash
python main.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动视角 |
| 鼠标点击角色面板 | 查看 NPC 详情 |
| 鼠标滚轮 | 滚动角色列表 |
| ESC | 退出游戏 |

## NPC 关系系统

| 聊天次数 | 关系等级 | 行为 |
|----------|----------|------|
| 0 – 2 次 | 陌生人 | 偶遇时随机打招呼 |
| 3 – 9 次 | 熟人 | 使用更亲近的日常用语 |
| 10 次以上 | 朋友 | 主动靠近对方，互相打电话 |

## 项目结构

```
SimWorld/
├── main.py              # 主入口，游戏循环，道路与汽车定义
├── constants.py         # 全局常量
├── entities/
│   ├── npc.py           # NPC 类，AI 行为与社交逻辑
│   └── car.py           # 汽车类，在主干道上行驶
├── world/
│   └── buildings.py     # 建筑定义与像素风绘制
└── ui/
    ├── draw_utils.py    # 摄像机、地面、道路、汽车、人物绘制
    └── hud.py           # HUD、小地图、角色面板
```

## 技术栈

- Python 3
