# SimWorld

一个用 Python + Pygame 构建的 2D 虚拟世界模拟器，拥有昼夜系统、NPC 社交关系与记忆机制。

## 功能特性

- **昼夜循环** — 天空颜色、太阳/月亮位置随时间动态变化，夜晚有遮罩加深效果
- **玩家控制** — WASD / 方向键移动，靠近物品自动拾取食物和水
- **饥饿 & 口渴系统** — 属性持续衰减，需要主动寻找资源补充
- **NPC AI** — 10 个有名字的 NPC 自主游走，会主动寻找食物/水，并在靠近玩家时赠送食物
- **社交关系系统** — NPC 之间通过聊天积累好感度，分为陌生人 / 熟人 / 朋友三个等级，朋友会主动靠近对方发起对话
- **差异化对话** — 不同关系等级触发不同的聊天内容
- **名字标签** — 每个 NPC 头顶显示姓名，状态异常时显示警示点

## 快速开始

**依赖**

```bash
pip install pygame
```

**运行**

```bash
python game.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动玩家 |
| 靠近食物/水 | 自动拾取 |
| ESC | 退出游戏 |

## NPC 关系系统

| 聊天次数 | 关系等级 | 行为 |
|----------|----------|------|
| 0 – 2 次 | 陌生人 | 偶遇时随机打招呼 |
| 3 – 9 次 | 熟人 | 使用更亲近的日常用语 |
| 10 次以上 | 朋友 | 主动靠近对方，使用亲密话语 |

## 技术栈

- Python 3
- Pygame
